# gboot
gboot is a small graphical calculator made in x86 Assembly <br /><br />
![](scrshot.png)
![](scrshot2.png)

# how to use
Use qemu/vbox/vmware/etc to run the gboot.img

# commands
- Addition: +
- Subtraction: -
- Multiplication: *
- Division: /
- Xor: ^
- And: &
- Or: |
- Shift right: >
- Shift left: <

Use the variables x and y to do operations
